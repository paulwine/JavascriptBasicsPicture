Javscript Basics

JavaScript is a cross-platform, object-oriented scripting language, JavaScript is extremely popular for a variety of reasons. It is a small and lightweight language allowing maximum flexibility for developers to take it in a bunch of different directions. JavaScript lives inside a host environment (a web browers or Node server), it can be connected to the objects of these environments to provide programmatic control over them.

  • Variable Declarations

    JavaScript variables are containers for storing data values - imagine a cupt you fill with coffee, the cup holds the coffee, a variable holds a value. All JavaScript variables must be indetifies with unique names. These unique names are called identifiers,

    var x;
  • Variable Assignment

    Assignment operators assign values to JavaScript variables - our cup can now have coffee poured in it, giving our variable a value to hold. The = assignment operator assigns a value to a variable,

    var x = 10;
